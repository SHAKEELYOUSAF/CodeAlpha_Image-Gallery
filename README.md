# Image Gallery - Task 01 (CodeAlpha Internship)

This is a responsive Image Gallery project developed as part of the CodeAlpha Frontend Development Internship.

It is built using HTML, CSS, and JavaScript and includes interactive features like navigation buttons, lightbox view, hover effects, smooth transitions, and responsive design for all screen sizes.

## Features
- Responsive image gallery layout  
- Next and Previous navigation  
- Lightbox image preview  
- Hover effects with smooth transitions  
- Mobile-friendly design  
- Optional categories/filters (bonus feature)

## Technologies Used
HTML, CSS, JavaScript 

## Project Purpose
The goal of this task is to practice frontend development skills including UI design, DOM manipulation, and responsive web design.
